# ILYASEMENOV — Japanese Wikipedia Word Frequency

## Source
- **GitHub:** https://github.com/IlyaSemenov/wikipedia-word-frequency
- **Cleaned fork:** https://github.com/adno/wikipedia-word-frequency-clean
- **Data date:** August 29, 2022 (jawiki dump)

## Description
Word frequency data generated by processing the Japanese Wikipedia dump using `wikiextractor`. The frequency metric is **document frequency** — the number of Wikipedia articles in which a word appears (not total occurrence count). Words appearing in fewer than 3 different articles are excluded.

Methodology:
- Strips punctuation, normalizes unicode
- Discards words containing digits
- Only includes words appearing in 3+ different articles
- Results cover ~1.7 million unique words for Japanese

## Format
Plain text, space-separated: `word document_count`. No header. Pre-sorted by document count descending.

## File Used
`jawiki-2022-08-29.txt` (~1.7M entries)

## DATA.csv
Top 25,000 words by document frequency. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

# JITEN_VISUAL_NOVEL — Visual Novel Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on Japanese visual novels from jiten.moe. Contains ~224,066 entries covering script text from a variety of visual novel titles.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

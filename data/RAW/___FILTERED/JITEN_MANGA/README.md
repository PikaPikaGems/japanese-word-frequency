# JITEN_MANGA — Manga Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on manga from jiten.moe. Contains ~264,038 entries covering a broad range of manga titles and genres.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

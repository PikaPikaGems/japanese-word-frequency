# JITEN_WEB_NOVEL — Web Novel Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on Japanese web novels from jiten.moe. Contains ~59,617 entries covering user-published fiction from web platforms.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

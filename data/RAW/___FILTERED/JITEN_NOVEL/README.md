# JITEN_NOVEL — Novel Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on Japanese novels from jiten.moe. Contains ~284,755 entries covering published fiction novels across various genres.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

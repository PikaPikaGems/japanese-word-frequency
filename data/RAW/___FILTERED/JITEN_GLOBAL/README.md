# JITEN_GLOBAL — Global (All Media Combined) Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list aggregated across all media types from jiten.moe. Contains ~430,669 entries combining anime, drama, manga, movies, novels, video games, visual novels, web novels, and more.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

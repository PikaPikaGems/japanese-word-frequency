# JITEN_NON_FICTION — Non-Fiction Media Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on non-fiction media from jiten.moe. Contains ~89,213 entries covering documentaries, educational content, and other non-fiction material.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

# JITEN_DRAMA — Japanese Drama Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on Japanese live-action drama from jiten.moe. Contains ~217,924 entries covering a wide range of drama productions.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

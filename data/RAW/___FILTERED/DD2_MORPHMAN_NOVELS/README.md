# DD2_MORPHMAN_NOVELS — Morphman Japanese Novels Frequency List

## Source
- **Creator:** Dave Doebrick (compiled), Morphman format
- **Collection:** Dave Doebrick's Frequency List Compilation
- **Original document:** https://docs.google.com/document/d/1IUWkvBxhoazBSTyRbdyRVk7hfKE51yorE86DCRNQVuw/edit

## Description
Word frequency list derived from a Japanese novel corpus (Kindle 5k), processed through Morphman.
Contains ~126,497 entries. Sorted by occurrence count descending.

## Format (`Morphman JapFreqList_5109_Novels.txt`)
Tab-separated, 2 columns: `occurrence_count`, `word`.

## DATA.csv
Top 25,000 words by occurrence count. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

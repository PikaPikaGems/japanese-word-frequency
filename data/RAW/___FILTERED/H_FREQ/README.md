# H_FREQ — Adult (18+) Japanese Content Word Frequency (Kuuube)

## Source
- **Kuuube's collection:** https://github.com/Kuuuube/yomitan-dictionaries

## Description
Word frequency list derived from Japanese adult (18+/hentai) content. The filename `h_freq_list.tsv` uses "h" to refer to adult content in Japanese internet terminology.

Contains ~44,712 entries. Format: tab-separated `word\toccurrence_count`, pre-sorted by count descending (no header). Top words: 私 (108,005), 言う (39,755), 中 (39,193), etc., alongside content-specific vocabulary.

**⚠️ Note:** This corpus is derived from adult content and includes 18+ vocabulary.

## Format
TSV: `word\toccurrence_count` (tab-separated, no header, pre-sorted by count descending).

## DATA.csv
Top 25,000 words by occurrence count. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

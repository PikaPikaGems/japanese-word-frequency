# HINGSTON — University of Leeds Japanese Word Frequency

## Source
- **GitHub:** https://github.com/hingston/japanese
- **Original corpus:** http://corpus.leeds.ac.uk/frqc/internet-jp.num

## Description
A list of the most common Japanese words in order of frequency, as determined by the University of Leeds Corpus. The Leeds corpus is a representative internet-based corpus covering multiple languages.

## Format
Plain text file (`44998-japanese-words.txt`) with one word per line, already ordered by frequency descending (line 1 = most frequent, rank 1). Contains ~44,998 entries.

## DATA.csv
Top 25,000 words by frequency order (line position). Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

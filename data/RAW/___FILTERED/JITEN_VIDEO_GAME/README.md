# JITEN_VIDEO_GAME — Video Game Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on Japanese video games from jiten.moe. Contains ~158,547 entries covering dialogue and text from a wide range of game titles.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

# JITEN_AUDIO — Audio Media (Podcasts/Audio Dramas) Frequency from jiten.moe

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on audio media (podcasts and audio dramas) from jiten.moe. Contains ~8,437 entries derived from transcribed audio content.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

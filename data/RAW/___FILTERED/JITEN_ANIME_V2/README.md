# JITEN_ANIME_V2 — Anime Word Frequency from jiten.moe (CSV format)

## Source
- https://jiten.moe/other
- https://github.com/Sirush/Jiten

## Description
Word frequency list based on anime from jiten.moe. Contains ~215,862 entries. This is a distinct source from JITEN_ANIME, which was processed from a Yomitan JSON export; this file is the direct CSV download (`frequency_list_Anime.csv`) with columns `Word, Form, Rank`.

## DATA.csv
Top 25,000 words by frequency rank. Columns: `WORD`, `FREQUENCY_RANKING` (1 = most frequent).

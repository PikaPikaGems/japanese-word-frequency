# ABOUT

This list was created by https://www.japanese-like-a-breeze.com. It was computed from 301 anki decks (mostly anime, some dorama), 1.848.163 flashcards / sentences where used. The deck source is https://japanesedecks.blogspot.com. I'm using this list is to choose words for my beginner deck: https://ankiweb.net/shared/info/911122782 Note, that the readings are incomplete, I only added those of the first entry in jmdict. Moreover, they can contain outdated forms of the expression. Relevant for the frequency is only the "Expression"-column. For technical details and error sources refer to this blog post: https://www.japanese-like-a-breeze.com/computing-a-word-frequency-list-based-on-anime/ For an interesting discussion, refer to: https://www.reddit.com/r/LearnJapanese/comments/ilobcq/word_frequency_list_based_on_anime_and_some/
